# Pagina-de-receita

Projeto:
Página web estática de uma receita de capcake de café com chantilly.

Tecnologias:
- HTML5
- CSS3

Recursos incluídos:
Estrutura organizada com listas de ingredientes e modo de preparo;
Design limpo;
Estilização personalizada (cores, fontes e espaçamentos harmoniosos).
